
# 🧾 Simple Billing System – PF (C Language)

## 📌 Overview

This project is a **Simple Inventory and Billing System** developed in **C language**. It is designed to manage stock items and generate customer invoices efficiently. It is suitable for **small-scale business use and academic learning purposes**.

## 🌟 Features

* 🔐 **Secure Login System** (3 attempts, password protected)
* 📦 **Inventory Management** (Add, View, Edit, Delete items)
* 🧾 **Invoice Generation** with automatic stock updates
* 💸 **Optional Discount Support**
* 📝 **File Handling System**

  * `inventory.txt` – Stores product data
  * `invoices.txt` – Stores billing history
* 🌐 **Cross-platform support** (Windows & Linux compatible screen clearing)
* ✅ **Input Validation** (quantity, price, phone number)
* 📆 **Real-time Date & Time display**
* 🧠 Built using standard C libraries:

  * `<stdio.h>`, `<stdlib.h>`, `<string.h>`, `<time.h>`, `<ctype.h>`

---

## 📁 Project Structure

| File Name       | Purpose                              |
| --------------- | ------------------------------------ |
| `main.c`        | Complete source code of the system   |
| `inventory.txt` | Stores inventory records             |
| `invoices.txt`  | Stores generated invoices            |
| `temp.txt`      | Temporary file for updates/deletions |
